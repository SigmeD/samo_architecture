## Назначение

Куратор франшиз (`op-kurator-franshiz-dnm`) — специалист головного офиса, который сопровождает и поддерживает франчайзи-партнёров (владельцев школ) от лица франчайзера: связующее звено «головной офис ↔ партнёры», ведёт от нескольких до пары десятков точек. Главная задача — **переводить убыточных/проблемных франчайзи в успешные**: практик, мотиватор и управленец, а не наблюдатель с отчётами. Основной акцент кабинета — **аналитика и дашборды**: рейтинги, план/факт, динамика «было → стало».

Прямое подчинение руководителю проекта (`lead`), адресат отчётов по созвонам. Куратор франшиз ≠ руководитель сети (руководитель сети — грань кабинета франчайзи-партнёра, не данный кабинет). Курирует франшизы целиком: запуск точек, операционная поддержка, аудиты стандартов, созвоны по графику, подключение экспертов, рейтинги сети, документы/регламенты. Финансы франчайзи — только чтение (для аналитики), без операционного управления школой (это контур франчайзи-партнёра).

## Ключевые инварианты (не нарушать)

- **Финансы франчайзи — ТОЛЬКО чтение** (для аналитики); куратор финансы школы не ведёт.
- **Долю роялти «50%» НЕ показывать** нигде.
- **Персональные имена нигде — только роли**; всё обезличено.
- **Балльной/grade-оценки учеников НЕТ.**
- Куратор франшиз — **прямое подчинение руководителю проекта** (`lead`).
- **Не дублирует операционное управление школой** (это кабинет франчайзи) — только сопровождение.
- Канон ВЫРОВНЕН 06.06: роль = аккаунт-менеджер франшизной сети (CONV-USER-ROLES v1.3 §9.3, SPEC-M3 v2.7 §3.12, br8 слита) — **не divergent**, расхождений с каноном нет.

## Домены и функции (14)

### 📊 Дашборд франшиз · _только просмотр_
Сводка: число франчайзи / школ, распределение по стадиям (30 / 60 / 90 / 120 / 150+ учеников) · Блок «Требует внимания»: убыточные, отставание план/факт, нет созвонов, проваленные показатели · Ближайшие созвоны + просроченные отчёты · Топ-рейтинги сети; «моя статистика» куратора.
_Источник: проектный документ ЛК §3.1._

### 🏫 Франчайзи и школы · _только просмотр_
Список с фильтрами: стадия · регион · проблемность · дата договора · Карточка школы: профиль / контакты / территория; показатели (ученики / группы / часы / продажи) · Финансы школы — ЧТЕНИЕ: приход / уход / роялти / маржа, план/факт · Персонал (ст. куратор / кураторы / продавцы с рейтингами); история; задачи; документы.
_Источник: проектный документ ЛК §3.2._

### 🚀 Запуск новых точек
Чек-листы открытия: этапы / прогресс / сроки / ответственные · Подбор и оформление помещения по стандартам бренда; закупка оборудования · Обучение персонала школы; присутствие на открытии (лично / дистанционно) · Стадии выхода: 30 → 60 → 90 → 120 → 150 учеников.
_Источник: Проектный док ЛК «Куратор франшиз» + встреча 05.06._

### 🛟 Операционная поддержка и обращения
Постоянная связь с партнёрами; закрепление за группой точек · Консультации: кадры, расписание, нестандартные ситуации, ценообразование · Входящие обращения / тикеты от франчайзи: статусы, SLA · Сопровождение, а не операционное управление школой (это контур франчайзи).
_Источник: Проектный док ЛК «Куратор франшиз» + встреча 05.06._

### ✅ Аудиты и контроль стандартов
Создание аудита по шаблону (чек-лист); плановые и внезапные проверки · Фиксация нарушений с фото; оценка соответствия стандартам сети · Соблюдение оформления / ассортимента / качества сервиса / стандартов бренда · План устранения нарушений и контроль закрытия.
_Источник: Проектный док ЛК «Куратор франшиз» + встреча 05.06._

### 📅 Созвоны и календарь
График: еженедельно общий + раз в 2 недели по направлениям (маркетинг/SMM · таргет · продажи · кураторы · финансы) · Создание созвона: все / сотрудник / школа; тема / дата / время · Авто-уведомление + ссылка + календарь; синхронизация календарей куратора и франчайзи · Хранение записей созвонов.
_Источник: SPEC-DNM-TZ-001 v3.8 §18.10; проектный документ ЛК §3.3._

### 🎓 Эксперты по направлениям
Эксперты (кураторы / продажи / финансы) с планом тем (напр. 6 встреч / 3 мес = 6 тем) · Запуск через Бизнес-академию ДНМ · Расписание встреч эксперта в общий календарь · Контроль прохождения и закрытия дефицита по аналитике.
_Источник: проектный документ ЛК §3.4 + встреча 05.06._

### 📄 Отчёты и фиксация
Отчёт по созвону: тема из названия · «что обсуждали» · «выводы» · Адресат — руководитель проекта (прямое подчинение); скачать / переслать франчайзи · Все свои отчёты + своя статистика в одном месте · ИИ-черновик допустим, без автозамены.
_Источник: проектный документ ЛК §3.5; CONV-RBAC-DNM-001 v1.4._

### 🔬 Аналитика · _только просмотр_
Сравнение точек между собой; динамика по периодам; причины спадов · Выгрузка данных; аналитика любой курируемой школы · Ученики / группы / часы / посещаемость / продажи — чтение · Финансы школы (приход / уход / роялти / маржа) — ЧТЕНИЕ для аналитики.
_Источник: проектный документ ЛК §3.6._

### 🎯 План / факт
Франчайзи сам ставит план (год / месяц: заполняемость, продажи) → виден куратору · Дашборд план vs факт по месяцам · При отставании — план действий → задачи / созвоны → контроль динамики · Динамика «было → стало» по курируемым школам.
_Источник: SPEC-DNM-TZ-001 v3.8; проектный документ ЛК §3.7._

### 🩺 Курирование (фокус на проблемных)
Зона риска: убыточные, ниже порога 60 учеников, отставание от плана, низкие рейтинги кураторов · По каждому: причина → план вывода в плюс → созвоны / эксперты → контроль закрытия · Динамика «было → стало» наверх (руководителю проекта) · Главная задача — перевести проблемных франчайзи в успешные.
_Источник: проектный документ ЛК §3.8._

### 🏆 Рейтинги сети · _только просмотр_
Рейтинг франчайзи: по числу учеников И по выручке · Рейтинги школ, кураторов, маркетологов, продавцов · Действие поверх: выбор лучших → отметка / мотивация → трансляция опыта (кейсы, приглашение на общий созвон) · Худшие → в зону курирования.
_Источник: SPEC-DNM-RATING-001 v1.2; проектный документ ЛК §3.6._

### 📚 Документы, регламенты и доступы
Хранилище пакетов для франчайзи: договоры, руководства, методички, условия франшизы · Загрузка регламентов, правовых документов, постановлений · Выдача и управление доступами к ЛК франчайзи-партнёров (провижининг кабинетов партнёров) · Адресная выдача документа партнёру → виден в его ЛК; версионирование.
_Источник: Проектный док ЛК «Куратор франшиз» + встреча 05.06._

### 🧑‍🎓 Моё обучение (Академия) · Чат · Задачи

## Модули (бизнес-процессы, 6)

1. **Цикл курирования франчайзи** _(partial)_ — Ядро роли (П1): Анализ → Созвон → Фиксация → Действие → Контроль план/факт → Развитие → повтор. Перевод проблемных франчайзи в успешные; «было → стало» наверх руководителю проекта.
2. **Плановый созвон по направлению** — П2: выбор направления (маркетинг/таргет/продажи/финансы/кураторы) → создание созвона → авто-уведомление / ссылка / календарь → запись → отчёт руководителю проекта.
3. **Подключение эксперта по направлению** — П3: дефицит по аналитике → эксперт + план тем (напр. 6 встреч / 3 мес) → курс через Бизнес-академию → расписание в общий календарь → контроль прохождения.
4. **Работа с рейтингами сети** — П4: рейтинги (франчайзи по числу учеников и по выручке, школы, кураторы, маркетологи, продавцы) → лучшие (мотивация / трансляция опыта) / худшие (зона курирования).
5. **Запуск новой точки** — Онбординг франчайзи: чек-лист открытия (помещение по стандартам бренда, оборудование, обучение персонала, открытие) и стадии выхода 30 → 60 → 90 → 120 → 150 учеников.
6. **Аудит стандартов сети** — Контроль стандартов бренда: аудит по шаблону (чек-лист) → плановая / внезапная проверка → фиксация нарушений с фото → оценка соответствия → план устранения и контроль закрытия.
