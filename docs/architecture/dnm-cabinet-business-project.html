Бизнес-проект — из ученика в предпринимателя

Финальный модуль старшей группы · что проживает подросток в системе ДНМ
старшая группа 14–17 · по желаниюсеместр 4 · финальный модульэкосистема SAMO Global
🎯
Цель модуля — не «сдать форму», а прожить путь владельца дела: своя идея → своя экономика → своя защита → реализация. Психолог снимает страх на каждом шаге, педагог переводит «взрослые» термины на язык подростка.
🚀 Арка трансформации — ядро модуля сдвиг самоощущения «ученик → предприниматель»
🧩 «Я ученик»→ 💡 «Есть идея»→ 🧮 «Считаю деньги»→ 🎁 «Упаковываю»→ 🤝 «Веду команду»→ 🏆 «Я предприниматель»
Фаза 1
💡 Есть идея
🏗 Шаг в системе
Идея. Тип (индивид./групповой), название, «идея в одном предложении», описание. Создаётся черновик.
🧠 Психолог
Барьер «моя идея глупая, засмеют» → черновик виден только мне, переписать можно сколько угодно.
🎓 Педагог
Вводит ценностное предложение: кому помогаем / что даём / чем отличаемся. Даёт рыбу-пример.
→
Фаза 2
🧮 Считаю деньги
🏗 Шаг в системе
Финансовая модель. Стартовый капитал, выручка, расходы, прибыль, окупаемость, период (в тенге).
🧠 Психолог
«Я не математик, цифры страшно» → поля по одному, авто-подсказка: прибыль = выручка − расходы.
🎓 Педагог
Объясняет термины капитала / выручки / расходов / прибыли / окупаемости + наглядный пример расчёта.
→
Фаза 3
🎁 Упаковываю
🏗 Шаг в системе
Презентация. Шаблон бизнес-плана, обязательные слайды, блок POAS, файл PPT/PDF до 50 МБ.
🧠 Психолог
«Не умею делать презентации» → готовый шаблон и чек-лист слайдов снимают «чистый лист».
🎓 Педагог
Термины: бизнес-план, защита, POAS (Plan of Action Summary) — структура слайдов как леса.
→
Фаза 4 · групповой*
🤝 Веду команду
🏗 Шаг в системе
Команда. Лидер приглашает одногруппников (до 5). *Для индивид. проекта — «сам себе лидер».
🧠 Психолог
Страх конфликта / «меня не возьмут» → роли понятны, лидер распределяет, вклад каждого виден.
🎓 Педагог
Термины: команда, лидер, роли, делегирование — первый опыт управления.
→
Фаза 5
🏆 Защищаю
🏗 Шаг в системе
Защита. Чек-лист готовности → отправка куратору → решение. Принят → +200 соларов.
🧠 Психолог
Страх сцены и «вернули = провал» → доработка это норма пути, наставник на твоей стороне.
🎓 Педагог
Термины: питч, обратная связь, итерация. «Пройдёшь — поможем реализовать по-настоящему».
Жизненный цикл проекта: черновик → на проверке → доработка ↔ принят. Возврат на доработку всегда с комментарием куратора — это часть обучения, а не приговор. За принятый проект ученику начисляется +200 соларов «за предприимчивость».
🧑‍🏫Куратор-наставник
  • Ведёт ученика по фазам, отвечает в чате группы
  • Проверяет проект, возвращает на доработку с комментарием или принимает
  • После принятия — помогает реализовать идею в жизни
🧠Психолог
  • Работает со страхом ошибки, сцены и сравнения с другими
  • Установка: «ошибка = опыт», «доработка = часть пути»
  • Поддержка при тревоге и выгорании в финальном модуле
🎓Педагог
  • Переводит «взрослый» бизнес-язык на язык подростка
  • Даёт примеры, рыбы-шаблоны и пошаговые леса
  • Следит, что каждый новый термин объяснён до того, как нужен
