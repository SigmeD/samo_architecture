Франчайзи видит агрегаты по всей сети и заходит в любую школу как директор (расширенная аналитика: финансы, ученики, посещаемость, кураторы, продажи). Иерархия: холдинг → франшиза → школа → группа → ученик.
🏫Сеть школ
  • 🏫Все школы франшизы — список + карточка каждой
  • 🎓Ученики всех школ — единый реестр сети
  • 👥Группы всех школ — учебные группы сети
⚙️Управление школами
  • 🎛️Настройка функций школ — вкл/выкл опц. модули (табель, звонки, органайзер) в кабинете админа
  • 🔎Вход в школу как директор — расширенная аналитика: финансы, ученики, посещаемость, кураторы, продажи
👔Персонал и обучение
  • 👔Сотрудники и партнёры · приглашение
  • 🎯Компенсации — оклад + бонусы по KPI
  • 🎓Конструктор обучения — кураторы, менеджеры, ученики
💰Финансы сети🔒 отчёты
  • 📊P&L, расходы, ФОТ — просмотр
  • 🏦Осн. средства, cashflow — просмотр
  • ↩️Возвраты по сети — рассмотрение
🛍️Магазин🪙 за солары
  • 📦Каталог товаров-призов — физ. / цифр.
  • 🚚Распределение и квоты по школам
  • 🏷️Остатки / инвентарь школ
📊Отчётность
  • 📈Отчёты: финансовый, сводка
  • 🏆Рейтинг франчайзи и школ
  • 🧑‍🏫По кураторам и созвонам
  • 🏠Дашборд (KPI, новые ученики)
📣Коммуникации
  • 🔔Уведомления
  • 📞Звонки + отчёты
