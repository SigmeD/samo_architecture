02

Роли и кабинеты

11 кабинетов в трёх контурах — у каждого свой набор функций и scope данных
Потребители
🎓
Кабинет ученика
ребёнок 10–17 лет · потребитель обучения
  • Уроки: смотрит заранее → на занятии разбор фрагментов с куратором
  • СУ (Система успеха) + Бизнес-проект — два отдельных блока
  • Звёздочки за активность → солары, уровни, рейтинг, магазин
  • ДЗ: статус «принято / не принято», средняя, успеваемость
  • Сертификат как ритуал: ролик «путь успеха» → Instagram → вручение
  • ИИ-аватар Довлатова, «Зачем учиться» (топ-100)
  • Расписание, чат группы, уведомления, профиль
  • 3 пробных урока на стадии знакомства с продуктом
👪
Кабинет родителя
контроль и оплата · отдельный аккаунт
  • Мои дети: статусы активен / не активен / ждёт оплаты · реф-ссылка
  • Центр контроля: мгновенное уведомление о непосещении урока
  • Сводка: ДЗ «принято/не принято», рейтинг; отчёты куратора (нед./месяц/семестр, видео-«кругляшочки»)
  • Старт с договора (онлайн-подпись) → оплата, рассрочка, график
  • Расторжение договора (поэтапно) · приостановка («академ»)
  • Чаты: куратор / админ / франчайзи · «поблагодарить» · опросы
  • Расписание ребёнка · приобретённые курсы · программа (видит все темы)
Операционная команда школы
🧑‍🏫
Куратор
ведёт группы · участвует в продаже
  • Уроки по процедуре занятия (реальная методика школы)
  • Проверка: посещаемость → ДЗ → успеваемость → классная
  • 3 цикла проверки: ДЗ · эссе · бизнес-проекты
  • Пробный урок (воронка продаж): коэф. успешности 70–80% встреча
  • Академия обучения куратора: квизы ~100, аттестация
  • Ежемесячный отчёт родителю · Мой KPI · рейтинг, топ-3 значок
  • Задачи и напоминания · встречи со ст. куратором (раз в 2 нед.)
🧑‍🏫
Старший куратор
играющий тренер · scope школы
  • Свои группы + собственная аналитика
  • Аналитика по подчинённым кураторам: этапы, средние, явка
  • Обучение новых кураторов · лидерборд кураторов
  • Перераспределение групп · приём месячных отчётов
🏫
Администратор школы
все операции своей школы
  • Маркетинг: лидогенерация, ролики, Instagram школы встреча
  • Продажи: воронка, пробные уроки, план наполнения (250/120) встреча
  • Зачисление = цепочка: лид → менеджер → пробный → договор
  • Учебный процесс: группы (лимит 5–8) / индивид / СУ / доп. / бизнес-проекты / выезды
  • Непосещение решает администратор; запись живых уроков
  • Отдел качества · штатка + оргструктура · дебиторка
🤝
Менеджер по продажам
привлечение и сделка
  • Лиды и воронка · 3 пути регистрации
  • Доступ к 3 урокам родителю → запись на пробный
  • Пробный урок с куратором → дожим в WhatsApp
  • Договор, рассрочка, график · дебиторка · KPI
📊
Финансист / бухгалтер
финконтроль · конфиденциально
  • P&L (ОПиУ), доходы/расходы, ФОТ, осн. средства, cashflow
  • Роялти 20% · план платежа · возвраты
  • Выплаты: Фикса + Бонус − Штрафы (KPI)
  • Финотчёты видят владелец + бухгалтер; один на сеть
🛡️
Качество и контроль
сквозная функция администрации
  • Работа с проблемными клиентами, сбор ОС (позитив/негатив)
  • Проверки кураторов по негативным отзывам
  • Записи живых уроков → оценка качества работы куратора
  • Опросы родителей = рейтинг школы
Сеть и управление
🧭
Куратор франшиз
закреплённые школы
  • Обучение кураторов и менеджеров закреплённых школ
  • Созвоны + ручные отчёты руководителю франшизы
  • Документы франшизы (чтение) · не работает с учениками
🏬
Франчайзи / директор
управление школой(ами)
  • Видит как директор: расширенная аналитика школы
  • Все функции админа + KPI / табель / рейтинг / явка в цифрах
  • 2 модели оргструктуры (1 школа / сеть школ)
  • Настройка прав, фиче-тоглы, ценовые диапазоны
🎯
Руководитель проекта+ подразд. опц.
вся сеть школ
  • Сводный дашборд всех школ · конструктор программы
  • Магазин мерча (каталог / распределение) · документы франшизы
  • Назначение кураторов франшиз · делегирование прав
  • KPI: школа → головной офис
