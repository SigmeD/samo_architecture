# UI Style Guide — САМО

Руководство по фирменному стилю для разработки интерфейсов платформы «Дети на миллион».
Источник: `docs/brandbook/brandbook-samo.pdf`

---

## Фирменные цвета

### Основной цвет

| Название | HEX | RGB | Применение |
|----------|-----|-----|------------|
| **SAMO Blue** | `#0055A7` | `rgb(0, 85, 167)` | Основной цвет бренда. Используется для primary-элементов: кнопки, заголовки, навигация, ссылки, иконки |

### Дополнительные цвета

| Название | HEX | RGB | Применение |
|----------|-----|-----|------------|
| **SAMO Orange** | `#F7933C` | `rgb(247, 147, 60)` | Акцентный цвет. Только как второстепенный: badges, уведомления, выделения, hover-эффекты |
| **SAMO Green** | `#00A550` | `rgb(0, 165, 80)` | Акцентный цвет. Только как второстепенный: статусы успеха, прогресс, подтверждения |

### CSS-переменные (рекомендация)

```css
:root {
  --color-primary: #0055A7;
  --color-accent-orange: #F7933C;
  --color-accent-green: #00A550;
  --color-white: #FFFFFF;
  --color-black: #000000;
}
```

## Типографика

### Основной шрифт — Roboto

Используется для всего основного текста интерфейса: заголовки, параграфы, формы, кнопки.

- **Roboto Regular (400)** — основной текст
- **Roboto Bold (700)** — заголовки, акценты

### Вспомогательный шрифт — Fira Sans

Используется для дополнительных элементов: подписи, метки, вторичная навигация.

- **Fira Sans Regular (400)**
- **Fira Sans Bold (700)**

### Подключение (Google Fonts)

```html
<link href="https://fonts.googleapis.com/css2?family=Roboto:wght@400;700&family=Fira+Sans:wght@400;700&display=swap" rel="stylesheet">
```

```css
body {
  font-family: 'Roboto', sans-serif;
}
```

## Логотип

- Логотип САМО — треугольник (синий/оранжевый/зелёный) + текст «САМО» + подпись «Международный центр развития человека»
- Название «САМО» — графическая форма (не шрифт), слоган — шрифт Franklin Gothic
- **Варианты**: цветной, ч/б, инверсный (белый на тёмном фоне), на фирменном синем фоне

### Правила использования

- Соблюдать охранное поле вокруг логотипа (минимум = ширина буквы «x» логотипа с каждой стороны)
- **Запрещено**: искажать пропорции, сжимать, растягивать, применять эффекты (тени, градиенты, обводки)
- На синем фоне (`#0055A7`) — использовать цветной вариант с белым текстом
- На чёрном фоне — использовать инверсный (полностью белый) вариант

## Принципы UI-дизайна (для платформы ДНМ)

1. **Основной цвет интерфейса** — SAMO Blue (`#0055A7`): шапка, навигация, primary-кнопки
2. **Оранжевый и зелёный** — только акценты, НЕ использовать как основной фон секций
3. **Шрифт интерфейса** — Roboto, Fira Sans как вспомогательный
4. **Детский контур** (кабинет ребёнка) может использовать более яркие оттенки оранжевого и зелёного, но синий остаётся доминирующим
5. **Контраст**: текст на синем фоне — всегда белый; текст на белом фоне — тёмно-серый/чёрный
